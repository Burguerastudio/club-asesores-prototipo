# Club de Asesores — Prototipo

Prototipo navegable (UX/UI) de **Club de Asesores** (universo *La Fortaleza del Asesor*), academia gamificada para el Club de la Asesoría. Cliente de **Sepia Creativa**; prototipo por **Burguera Studio**. El desarrollo final lo hace el equipo de desarrollo del proyecto; este repositorio es la referencia y este README el documento de entrega.

> Estado: arranque. El prototipo se construye por fases. Este README se completa con el universo, el mapa de páginas, el sistema de diseño y la gamificación a medida que avanza el build.

## Cómo correr el prototipo

Proyecto Astro. Requiere Node 18 o superior.

```bash
npm install
npm run dev      # servidor de desarrollo
npm run build    # genera la versión de producción en dist/
npm run preview  # sirve la build localmente
```

La presentación de las dos direcciones visuales (Fase 3) está en `src/pages/index.astro`.
Las imágenes de referencia viven en `public/placeholder/` y la iconografía en `public/icons/`.
